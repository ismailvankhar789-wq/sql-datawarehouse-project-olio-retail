# 🛒 Olist Retail Analytics Data Warehouse Project

## 📌 Project Overview

This project demonstrates the design and implementation of a modern data warehouse for the Olist Brazilian E-commerce dataset. The project follows industry-standard ETL practices by transforming raw transactional data into a clean, analytics-ready star schema for business intelligence and reporting.

The complete workflow includes:

- Bronze Layer (Raw Data)
- Silver Layer (Data Cleaning & Standardization)
- Gold Layer (Dimensional Data Warehouse)
- SQL Business Analysis
- Power BI Dashboard
- Business Insights & Recommendations

---

## 🎯 Project Objectives

- Build an end-to-end SQL data warehouse
- Clean and transform raw retail data
- Design a dimensional star schema
- Create optimized fact and dimension tables
- Perform exploratory business analysis using SQL
- Build an interactive Power BI dashboard
- Generate actionable business insights
